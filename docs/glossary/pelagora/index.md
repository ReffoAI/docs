# Pelagora

Pelagora is an open, decentralized peer-to-peer commerce network. It defines the shared protocols, data models, and networking layer that allow independent [Beacons](/glossary/beacon/) to discover each other and exchange inventory.

- **[Ref](/glossary/pelagora/ref)** — Refs, traits, categories, and statuses
- **[Web UI](/glossary/pelagora/web-ui)** — The built-in browser interface for managing inventory
- **[DHT Networking](/glossary/pelagora/dht-networking)** — Peer discovery and communication over Hyperswarm
